## NodeJS com Banco Relacional PostgreSQL (instância RDS da AWS).

## Objetivo
Desenvolver uma API, com Sequelize, integrando SQL ao Node.js.

## Descrição
Esse projeto integra o Node.js à um BD SQL. O BD utilizado na implementação do código foi o PostgreSQL.

API utilizando requisitos REST.

## Recursos Utilizados
Docker, com imagem do Postgres.

Posteriormente configurei uma instância do Postgres RDS da AWS 

Postbird.

Insomnia.
